# Project Name

3. Player animations
4. Improved map maker
5. Sound effects and music

### Long term
1. After finishing engine, plan to develop Pokemon inspired game

## Installation

1. Clone the repository:
    git clone https://github.com/ozhang05/pokemon.git
    cd pokemon
2. Build project
    mkdir build
    cd build
    cmake ..
    make pokemon
3. Running project
    ./pokemon


## Useage

WASD Controls
P to enable Debug mode
